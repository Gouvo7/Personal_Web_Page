Projects

Throughout my journey as a developer, I have worked on a variety of projects that have helped me gain the experience and knowledge needed to become a successful developer. Some of the projects I have worked on include:



Vodafone Appointment Manager



Built using Java, this application was used in my time working for Vodafone Group as a salesman and tech specialist. The company was tracking down appointments on a daily basis and a monthly report was mandatory. The capabilities of this application are that users can arrange appointments with customers, edit appointment information and produce daily and monthly reports through the app.



Shooting Range Management



A while ago, there was a shooting range organization in Athens that contacted me and a friend of mine and they told us about the various problems that the have came across. Basically, none of the procedures (registration, various paperwork, warehouse count, training/match statistics etc.) were digitized, and the members of the organization had to fill in papers and send them to the local authorities. This application idea would resolve all of those problems by making all of the above procedures automated. The project was soon dropped as the organization could not support it further, but it was very helpful on extending my portfolio (C#).



Exam Schedule Composition System (Thesis)



Built using Java and other libraries, this application was built as my thesis project. The scope is that this app would be used for the purposes of each final exam schedule by my university, and later on, it would be adapted across various universities all over Greece. In a summary, the app uses .xlsx files as input and output at the same time, reading the excel files upon starting the application and saving the changes in a file. For portability and maintenance reasons, the catch is that we did not want to use any databases and for this reason, this is the reason that we chose to use .xlsx files for reading and storing data.
